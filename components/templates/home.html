Welcome to RSIMU

A wonderful web app to get the most recent information about the stock data of the biggest companies in the USA.

RSIMU helps you to make decision on stock investment by visualizing the historical stock price

To get started click on any menu item on the left, such as Microsost Corporation. The stock information for that item is retrieved from the Quandle API, and visualized in two views: table and plot.

Currently the plot view compares two columns, "open price" vs "high price".

Implemented Features

In this project, the following features are implemented:

  • Authentication: Login, Register
  • Routing in a single page app
  • API connection using subscription
  • Data visualization in Table and Plot

This project is implemented using the most recent version of Angular 2. During the development of this project, Angular2 was still under beta version, and I needed to keep updating. However, I found it easy to learn and well-designed. The most interesting part for me is the how the concept of reactivity is implemented via "Observable" type, which I used in this project. Here are some of design goals and features of Angular 2 (compared to Angularjs)*:

  • Much improved performance
  • Improved modularity
  • Improved Dependency injection
  • Web component friendly
  • Support for Shadow DOM
  • Support for native mobile rendering in Android and iOs
  • Support for server side rendering
  • Improved testability
*According to this blog.
